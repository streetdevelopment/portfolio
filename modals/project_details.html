Features

User Side Features

Intuitive Landing Page

Clean and user-friendly interface with quick navigation to core features.

Interactive Campus Map

Clickable buildings with detailed information, descriptions, and multiple images. Toggleable map markers and building names for enhanced visibility.

Route Navigation

  • Step-by-step directions powered by Dijkstra's Algorithm for the shortest path. Autocomplete feature for starting and destination points.
  • GPS integration to use the current location as the starting point.

Map Legend

Provides clear descriptions of symbols and markers used on the map.

University Procedures

Easy access to step-by-step guidance for campus-related activities.

Events Section

Stay updated with university events, including details and locations.

Bug Reporting System

Simple form for users to submit feedback or report issues.

Admin Side Features

Comprehensive Admin Dashboard

  • Static map view.
  • Overview of map elements, procedures, events, and feedback summaries.

Path Management

Add, delete, or update navigation routes.

Building Management

Manage building boundary, entrypoints, and marker positioning, as well as building data, including images and descriptions.

Procedure Management

Add or update detailed steps for university processes.

Event Management

Add or modify university event details.

Office Management

Update office information for accuracy.

User Management

Create, delete, and manage user accounts.

Feedback Management

Review and manage user feedback for continuous system improvement.
